jQuery Introduction

jQuery is a fast, lightweight, and powerful JavaScript library that makes it easier to write JavaScript code.There are lots of other JavaScript libraries out there,but jQuery is probably the most popular,and also the most extendable.jQuery is use for Google,IBM,Netflix,Microsoft. The purpose of jQuery is make much easier to use Javascript on your website.jQuery follows the principle "Write Less, Do More."

We can do following in jQuery:-

  • HTML/DOM manipulation

  • CSS Manipulation

  • HTML event methods

  • Effects and Animations
